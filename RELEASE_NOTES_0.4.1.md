# FerroxPlorer 0.4.1 更新说明

本版本为稳定性与兼容性修复版本：

- 修复侧栏右键菜单混入压缩软件、Git、网盘等文件区扩展项的问题；侧栏现仅显示打开、新标签页、新面板和取消固定等导航操作。
- 优化受保护目录操作：新建文件/文件夹、重命名及移入回收站遇到拒绝访问时，会按需弹出 Windows UAC 请求管理员权限，无需以管理员身份运行整个应用。
- 修复空格预览中代码高亮多层文本基线不一致导致的文字重叠、错位问题，源码视图改为稳定的单层等宽排版。
- 修复无扩展名普通文件被错误识别为文件夹图标的问题，普通文件与文件夹现在使用独立的系统图标缓存和文件属性。
- 修复竖屏、旋转元数据及非方形像素视频在空格预览中被压缩的问题；视频按 Media Foundation 提供的显示宽高比等比居中播放。
- 视频预览新增全屏按钮；全屏播放保持原始比例，按 Esc 可先退出全屏，再关闭预览。

- `cargo test`：42 项测试全部通过
- Windows Release 构建与 Inno Setup 安装程序由 GitHub Actions 自动生成
